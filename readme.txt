A WooCommerce payment gateway plugin to integrate Daraza Payments for seamless and secure transactions.

== Description ==

The Daraza Payments Gateway plugin enables you to accept payments through Daraza directly in your WooCommerce store. It also allows administrators to manage remittances and wallet balances from the admin panel.

**Features include:**
- Secure payment processing via Daraza API.
- Wallet balance management.
- Remittance processing for administrators.
- Logging API requests and responses for debugging.
- Easy-to-use settings in the WooCommerce admin interface.

== Installation ==

1. Download the plugin.
2. Upload the plugin files to the `/wp-content/plugins/daraza-payments` directory or install it through the WordPress plugins screen directly.
3. Activate the plugin through the 'Plugins' screen in WordPress.
4. Navigate to WooCommerce > Settings > Payments > Daraza Payments to configure the plugin.
5. Add your Daraza API key in the plugin settings.

== Frequently Asked Questions ==

= What is required to use this plugin? =
You need an active WooCommerce installation and a valid Daraza API key.

= How do I obtain an API key? =
Contact Daraza support at [https://daraza.net](https://daraza.net) to request your API key.

= Can I manage remittances and wallet balances from the admin panel? =
Yes, the plugin includes features for admins to handle remittances and view wallet balances.
